# Tennis Toolkit 网球训练记录工具

一个简单好用的网球训练记录工具，支持云端数据同步。

## 功能特点

- ✅ 训练记录管理
- ✅ 技能进度追踪
- ✅ 问题与解决方案记录
- ✅ 云端数据同步（基于 Firebase）
- ✅ 跨设备访问
- ✅ 离线缓存支持

## 在线访问

访问地址：https://leepeixi.github.io/tennis-toolkit/

## 使用说明

### 本地运行
直接用浏览器打开 `index.html` 即可。

### 云端同步
1. 打开应用后，在"云端同步"标签页注册账号
2. 登录后数据将自动同步到云端
3. 在其他设备登录同一账号即可同步数据

### 同步的数据
- 学习路线和技能进度
- 技能笔记
- 训练日志
- 问题诊断记录
- 训练类型设置

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- 云端：Firebase (Authentication + Firestore)
- 部署：GitHub Pages
